# Airline Reservation Project

This is a **full-stack airline reservation application** built with:

- **Frontend:** React.js  
- **Backend:** Spring Boot (Java)  
- **Database:** mysql  

The project allows booking and managing flights through a user-friendly interface.

---

## Prerequisites

Before running the project, ensure you have:

- **Node.js** (for React frontend)  
- **Java 17+** (for Spring Boot backend)  
- **Git** (to clone the repository)

## Running the Backend (Spring Boot)


#### Windows
```cmd
cd AMS
mvnw.cmd spring-boot:run

```
for frontend 
```cmd
cd frontend
npm install
npm start
````
